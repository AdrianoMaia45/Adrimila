# Site Romântico - Nossa História

Este é um site single-page interativo e romântico criado para celebrar um relacionamento especial. Construído com HTML5, CSS3 moderno e JavaScript puro.

## Funcionalidades

- **Hero Section**: Cabeçalho de tela cheia com imagem de fundo e mensagem romântica.
- **Contador de Tempo**: Mostra o tempo exato desde o início do relacionamento (11 de abril de 2024).
- **Timeline de Amor**: Linha do tempo vertical com eventos importantes do relacionamento.
- **Galeria de Momentos**: Grid responsivo de imagens com efeito de hover.
- **Rodapé**: Mensagem carinhosa com coração pulsante.

## Design

- **Cores**: Paleta sofisticada com Rosa Quartzo (#F7CAC9), Off-White (#F5F5F5) e Dourado Champagne (#D4AF37).
- **Tipografia**: 'Playfair Display' para títulos e 'Montserrat' para textos.
- **Estilo**: Bordas arredondadas, sombras suaves (glassmorphism leve) e transições suaves.
- **Responsivo**: Layout adaptável para dispositivos móveis.

## Como Usar

1. Abra o arquivo `index.html` em um navegador web.
2. O contador começará automaticamente a contar o tempo desde 11 de abril de 2024.
3. Role a página para ver as animações de fade-in nos elementos.

## Personalização

- **Datas**: Ajuste a data de início no `script.js` (linha 6) se necessário.
- **Conteúdo**: Edite os textos e imagens no `index.html`.
- **Estilos**: Modifique cores e estilos no `style.css`.
- **Imagens**: Substitua os placeholders por suas próprias imagens.

## Tecnologias Utilizadas

- HTML5
- CSS3 (com Flexbox e Grid)
- JavaScript (ES6+)
- Google Fonts
- Animate.css
- Intersection Observer API
